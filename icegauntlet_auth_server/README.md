## How to run

Just run with:

./auth_server --Ice.Config=auth_server.conf

The _proxy_ will be shown in _stdout_. Also you can add users with:

./add_user <user_name>

./cliente.py < proxy >

- ChangePassword: Opción 1
  - introducir:
    - user
    - password
    - new password

- GetNewToken: Opción 2
	- introducir:
		- user
		- password
